# 🏥 Дарыгердин жардамчысы - Doctor Helper System

Медициналык мекемелер үчүн пациенттерди жана врачтарды башкаруу системасы. Кан басымын көзөмөлдөө, AI анализдөө, врачтарды башкаруу жана толук статистика.

---

## ✨ Негизги өзгөчөлүктөрү

### 👨‍💼 Администратор үчүн:
- Врачтарды кошуу, өчүрүү жана башкаруу
- Бардык врачтардын статистикасын көрүү
- Аудитория журналын көзөмөлдөө
- Резервдик көчүрмө алуу
- Толук отчеттор жана аналитика

### 👨‍⚕️ Врач үчүн:
- Өз кабинетине кирүү (логин/пароль)
- Пациенттерди каттоо жана башкаруу
- Кан басымын жана виталдык көрсөткүчтөрдү кошуу
- 🤖 AI менен басымдын абалын анализдөө
- Пациенттердин толук тарыхын көрүү
- Өз статистикасын көрүү
- Тренд анализи

## 🚀 Орнотуу жана ишке кошуу

### 1-вариант: Node.js менен (сунушталат)

```bash
# 1. Репозиторийди клондоо
git clone https://github.com/your-username/doctor-helper-system.git
cd doctor-helper-system

# 2. Керектүү пакеттерди орнотуу
npm install

# 3. Серверди ишке киргизүү
npm start

# же Development режими (автоматтык рестарт менен)
npm run dev
